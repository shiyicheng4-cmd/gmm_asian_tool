下载 & 运行

  1. 把本仓库根目录下的 index.html 提交到 GitHub。
  2. 按 README 下载 football-data 与 Understat 的 CSV，放入 data/ 路径；运行 python backtest.py 生成结果。
  3. 把 output/ 的图片/CSV 推到仓库（或放到 docs/ 目录供页面读取）。

你也可以在此处放一个「下载回测包」按钮，指向仓库里的 zip 或 Release。

如何在 GitHub 上开启 Pages（手机也可操作）

  1. 进入仓库 Settings → 向下滑到 Pages（有时在 “Code and automation” 分组下）。
  2. Source 选择：Deploy from a branch。
  3. Branch 选择：main，root（/），然后点击 Save。
  4. 等待约几十秒，访问：https://你的用户名.github.io/你的仓库名/。

如果你更喜欢把网页文件放到 docs/ 目录，也可以将 Pages 的目录设置改为 /docs。
